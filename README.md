###Git
一个快速了解git的网站：[git简明指南](http://www.bootcss.com/p/git-guide/)
本地仓库由工作目录、缓存区（index）和HEAD组成：
- 工作目录：实际持有的文件
- Index：临时保存本地的修改
- HEAD：指向最近一次提交后的结果

[Git 工作区、暂存区和版本库概念](http://www.runoob.com/git/git-workspace-index-repo.html)
